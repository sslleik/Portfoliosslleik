# 📊 TGK Analytics V2 - Раздельная система аналитики

## 📋 Общая архитектура

Аналитика разделена на **две части**:

### 1️⃣ **МГНОВЕННЫЕ ДАННЫЕ** (Instant Data)
Отправляются **один раз** при загрузке страницы

### 2️⃣ **ПОВЕДЕНЧЕСКИЕ ДАННЫЕ** (Behavioral Data)
Отправляются **регулярно** (каждые 30 сек) во время использования сайта

---

## 📤 ЧАСТЬ 1: МГНОВЕННЫЕ ДАННЫЕ

Собираются и отправляются **сразу после загрузки** (через 1-2 сек).

### 📋 1. Базовая информация

```
✅ IP-адрес
✅ Дата и время посещения
✅ Страна и город (по IP)
✅ Referrer (откуда пришёл пользователь)
✅ Текущий URL страницы
✅ Язык браузера
✅ Часовой пояс
```

### 📱 2. Данные устройства

```
✅ Тип устройства (ПК / Телефон / Планшет)
✅ Операционная система
✅ Браузер и версия
✅ User Agent
✅ Разрешение экрана
✅ Плотность пикселей (DPI)
✅ Количество ядер CPU
✅ Объём RAM
✅ Тип сети (4G / WiFi / Ethernet)
✅ GPU информация (WebGL)
```

### 🔑 3. Идентификация пользователя

```
✅ Статус cookies (включены/отключены)
✅ Количество cookies
✅ LocalStorage (включён/отключен)
✅ SessionStorage (включён/отключен)
✅ Размер LocalStorage
✅ Unique Session ID
```

### 🔐 4. Browser Fingerprint

```
✅ Canvas fingerprint
✅ WebGL информация (vendor, renderer)
✅ Список обнаруженных шрифтов
✅ Браузерные плагины
✅ Уникальный идентификатор
```

### 📨 Формат отправки мгновенных данных

**Отправляется как TXT файл** `instant-data.txt` с полной информацией параллельно с HTML сообщением.

Пример в Telegram:

## 📊 ЧАСТЬ 2: ПОВЕДЕНЧЕСКИЕ ДАННЫЕ

Собираются **во время использования сайта** и отправляются **каждые 30 секунд** + при закрытии страницы.

### 🎯 1. Данные поведения

```
✅ Время на странице (в реальном времени)
✅ Глубина прокрутки страницы (%)
✅ Движение курсора (координаты X, Y)
✅ Статус активности (активен/неактивен)
✅ Время последней активности
```

### 🖱️ 2. События (Event Tracking)

#### Клики:
```
✅ Клики по кнопкам (button click)
   - Текст кнопки
   - ID кнопки
   - Class кнопки
   - Время клика
```

#### Переходы по ссылкам:
```
✅ Клики по ссылкам (link click)
   - URL ссылки
   - Текст ссылки
   - Target (_blank / _self)
   - Время клика
```

#### Формы:
```
✅ Взаимодействие с формами
   - Тип поля (input / textarea / select)
   - Имя поля
   - Отправка формы (submit)
   - Количество полей в форме
```

#### Авторизация:
```
✅ Попытки входа
✅ Ошибки авторизации
✅ Регистрация пользователя
```

#### Файлы:
```
✅ Скачивание файлов
```

### 📈 3. Поведение сессии

```
✅ Переход между страницами
✅ Количество просмотренных страниц
✅ Длительность сессии (в реальном времени)
✅ Является ли пользователь новым или возвращающимся
```

### 📍 4. Завершение сессии

При уходе со страницы отправляется финальный отчёт:
```
📍 КОНЕЦ СЕССИИ
─────────────────────
Session ID: [ID]
Время на сайте: 125 сек
Последняя активность: 14:32:45
Глубина скролла: 73%
Всего кликов: 8
```

## 🔄 Специальные события

### Отслеживаемые действия:

1. **Клик по кнопке** → Регистрируется сразу

3. **Отправка формы** → Регистрируется сразу

## 📊 Статистика и метрики

Собираемые метрики позволяют анализировать:

✅ **Трафик**: Кто посещает, с каких устройств, из каких стран  
✅ **Поведение**: Как пользователи взаимодействуют с сайтом  
✅ **Engagement**: Время на сайте, глубина скролла, количество кликов  
✅ **Устройства**: ОС, браузеры, наиболее популярные комбинации  
✅ **Сессии**: Длительность, количество просмотров, возвращаемость  
✅ **Fingerprint**: Уникальность браузеров, обнаружение ботов  

---

## 🔒 Приватность и безопасность

- Данные отправляются только в ваш Telegram бот
- IP адреса используются только для геолокации
- Fingerprint не позволяет идентифицировать реального человека
- Можно отключить отправку в любое время (установить CHAT_ID: null)
